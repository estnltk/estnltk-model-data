## Data description

The benchmark contains sentences taken form <puupank> with manually curated syntax tree.

The sentences contain discourse phrases that were manually annotated to be 'free' (can be removed), 'bound' (can not be removed) and other.

Random sentences with free phrases and sentences with bound phrases are used to create the benchmark file.


## Additional annotations

The benchmark also contains additional annotations about location phrases. The annotation guides can be found [here](../labeling_guides/).
